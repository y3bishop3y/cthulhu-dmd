# Front Card Character Extraction Coordinates

For characters that have front cards, use these coordinates for better character extraction:

## Default Front Card Coordinates
- **X**: 55%
- **Y**: 3%
- **Width**: 37%
- **Height**: 90%

## Usage

### In `prepare_for_web_upload.py`:
```bash
--front-coordinates "55%,3%,37%,90%"
```

### In `create_gif.py`:
```bash
--coordinates "55%,3%,37%,90%"  # When using front card
```

## Back Card Coordinates (for reference)
- **X**: 15%
- **Y**: 17%
- **Width**: 28%
- **Height**: 58%
