# Game Rules Documentation

This directory contains comprehensive documentation of all Cthulhu: Death May Die game mechanics.

## Documents

### Core Mechanics
- **[Dice Mechanics](./dice-mechanics.md)** - Dice types, faces, symbols, probabilities
- **[Health & Stress](./health-stress.md)** - Health track, stress track, Rest action
- **[Insanity Track](./insanity-track.md)** - Insanity progression, red swirls, level-ups, tentacle risk
- **[Turn Structure](./turn-structure.md)** - Actions, turn phases, action economy

## Purpose

These documents serve as:
1. **Reference** - Quick lookup for game mechanics
2. **Implementation Guide** - Details for code implementation
3. **Analysis Foundation** - Rules needed for power analysis
4. **Documentation** - Living documentation of game rules
