# Documentation Directory

This directory contains comprehensive documentation for the Cthulhu: Death May Die game rules, mechanics, and analysis systems.

## Structure

```
docs/
├── README.md                    # This file
├── game-rules/                  # Core game mechanics documentation
│   ├── README.md
│   ├── dice-mechanics.md
│   ├── health-stress.md
│   ├── insanity-track.md
│   └── turn-structure.md
├── analysis/                    # Analysis system documentation
│   ├── README.md
│   ├── power-combination.md
│   ├── upgrade-optimization.md
│   └── synergy-analysis.md
└── examples/                     # Example analyses and use cases
    ├── README.md
    └── power-combinations.md
```

## Game Rules

The `game-rules/` directory contains detailed documentation of all game mechanics:
- **Dice Mechanics** - Dice types, faces, probabilities, symbols
- **Health & Stress** - Health track, stress track, Rest action
- **Insanity Track** - Insanity progression, red swirls, level-ups
- **Turn Structure** - Actions, turn phases, action economy

## Analysis

The `analysis/` directory contains documentation for the power analysis system:
- **Power Combination** - How powers combine and interact
- **Upgrade Optimization** - Optimal upgrade path calculation
- **Synergy Analysis** - Character team synergy detection

## Examples

The `examples/` directory contains practical examples and use cases:
- Power combination examples
- Upgrade path examples
- Team synergy examples
